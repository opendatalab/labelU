# 数据导入

:::note

一个任务下上传相同文件名的文件将会上传失败。

:::

## 图片导入

点击「上传文件」，文件类型包括jpg，png，bmp，gif，或「上传文件夹」按钮，右侧显示上传成功后，再点击「下一步」保存并进入「标注配置」页面。

**注：** 目前数据导入仅支持本地数据导入，支持文件或文件夹导入两种方式。单次上传文件最大数量为100个，单个文件大小不超过100MB。

另外，图片导入支持上传jsonl格式的预标注文件，参考[示例](/schema/pre-annotation/json)

![](/zh/image-import.png)

## 视频导入

点击「上传文件」，文件类型包括MP4，或「上传文件夹」按钮，右侧显示上传成功后，再点击「下一步」保存并进入「标注配置」页面。

另外，视频导入支持上传jsonl格式的预标注文件，参考[示例](/schema/pre-annotation/json)

![](/zh/video-import.png)

## 音频导入

点击「上传文件」，文件类型包括MP3，wav，ogg，或「上传文件夹」按钮，右侧显示上传成功后，再点击「下一步」保存并进入「标注配置」页面。

另外，音频导入支持上传jsonl格式的预标注文件，参考[示例](/schema/pre-annotation/json)

![](/zh/audio-import.png)

## 预标注导入

### JSONL 文件

通过 JSONL 文件导入，每一行是一个样本的 JSON 格式的标注信息，JSON 格式参考[预标注格式-JSONL](/schema/pre-annotation/jsonl/image)

### JSON （兼容 LabelU 导出的标注文件）

LabelU 导出的标注文件（或自行组装），可以通过JSON文件导入，json 格式参考[预标注格式-JSON](/schema/pre-annotation/json)
